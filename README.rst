rsanic
==========

Micro framework built on top of sanic.py written in Python 3.

Installing rsanic
=====================

.. code-block:: bash

    pip3 install rsanic

or

.. code-block:: bash

    git clone https://github.com/reformo/rsanic.git
    cd rsanic
    python3 setup.py

after install try example

.. code-block:: bash

    git clone https://github.com/reformo/rsanic.git
    cd rsanic/example
    python3 server.py
